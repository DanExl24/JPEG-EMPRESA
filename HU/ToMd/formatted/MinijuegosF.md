# Documentación de Historias de Usuario

## HU06: Mini-juegos para calentamiento

**Estado:** Tareas por hacer | **Prioridad:** Medium

### Historia de Usuario
**Como:** instructor
**Quiero:** un modulo de mini-juegos interactivos de parejas
**Para:** tener un modelo de juego en donde el usuario debe arrastrar las imágenes al texto correspondiente

### Criterios de Aceptación
- El mini-juego es un calentamiento, por cada sesión de juego hay 3 tarjetas con imágenes y poder arrastrar hacia el texto correcto, un máximo de 4 sesiones para calentar.
- Si la imagen no concuerda con el texto vuelve a su posición anterior a la que estaba.
- El mini-juego de ser arrastrar tarjetas de los tema de comunicación, por ejemplo: con imágenes del sol, la tarde y la luna hacia las palabras "Good morning", "Good afternoon", "Good evening".
- Tener un contador de cuantas sesiones lleva, son maximo 4 y con ello se debe visualizar.
- Al finalizar el mini-juego se muestra las sesiones respuestas correctamente y si tuvo algunas erróneas mostrarlas.
- Las preguntas que se respondieron mal él estudiante puede ver cual fallo para tener una retroalimentación.
