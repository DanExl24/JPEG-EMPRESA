# Documentación de Historias de Usuario

## HU03: Creación de actividades educativas

**Estado:** Tareas por hacer | **Prioridad:** Medium

### Historia de Usuario
**Como:** Instructor de inglés
**Quiero:** contar con herramientas para crear y configurar actividades interactivas
**Para:** enriquecer el contenido de cada módulo y evaluar el progreso de los aprendices de forma dinámica

### Criterios de Aceptación
- El sistema debe permitir la creación de actividades en base a plantillas predefinidas.
- Las plantillas de actividades predefinidas deben ser las siguientes:
- Sopas de letras
- Crucigramas
- Quizzes
- Preguntas de respuesta multiple
- Conectar significados.
- Actividades de escucha
- Actividades de pronunciación
- El sistema debe permitir que cada actividad creada se asigne a un módulo y a una fase específica (Preparación, Absorción, Práctica o Cierre), asegurando que aparezca en el lugar correcto del mapa de aprendizaje.
- Cada plantilla debe incluir campos fáciles de llenar según su tipo; por ejemplo, una lista de palabras para la sopa de letras o la pregunta con sus opciones correctas e incorrectas para los quizzes.
- Antes de guardar o publicar la actividad, el instructor debe poder ver una demostración de cómo la verá y la jugará el aprendiz para corregir posibles errores.
- El sistema debe permitir modificar o eliminar actividades creadas. Sin embargo, si un aprendiz ya resolvió la actividad, el sistema no debe permitir su edición.
- El instructor debe poder definir cuántos puntos vale la actividad o si es necesaria una nota mínima para que el sistema la marque como "completada" y permita seguir al siguiente paso.
- Se debe poder configurar un mensaje de felicitación si el aprendiz acierta y un mensaje de ayuda o refuerzo (pista) en caso de que cometa un error.
- El sistema debe permitir que el instructor decida si el aprendiz tiene intentos ilimitados o un número específico de oportunidades para completar la actividad con éxito.
